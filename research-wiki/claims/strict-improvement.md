# Strict recovery gain when phi is MI-insufficient

**status:** `drafted`

## Statement
If I(S; phi(X)) < I(S; X) (phi strictly loses target MI) then the Bayes-optimal estimator strictly beats the phi-restricted attack on a positive-probability set (orthogonality decomposition; linear-MMSE suboptimal unless jointly Gaussian).

## Honest scope
Avoid single-metric converse of sufficiency: use strict-MI-loss framing, NOT 'phi non-sufficient => strictly worse'. Requires finite second moment.
