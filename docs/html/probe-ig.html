Gaussian MI ceiling
— probe I_G

The Gaussian channel-capacity mutual-information ceiling: a closed-form converse on embedding-inversion leakage, computed from the embedding covariance spectrum and the release noise alone. It runs no inverter, so it bounds the recovery of every attack at once.

probe reference · 2026-06-24
§01

Algorithm

inputs → transforms → bits

Given the released channel Y = e₀ + N(0, σ²I) on a clipped embedding e₀ whose secret is the source text, I_G reads only the second-order geometry of e₀ and the noise scale σ.

  1. Estimate the covariance Σ = Cov(e₀) over the secret prior (the clean clipped embeddings of the corpus).
  2. Eigendecompose Σ to its eigenvalues λ₁ ≥ … ≥ λ_d (via a covariance eigendecomposition, not a full data SVD).
  3. Form the per-mode capacity tᵢ = ½ log₂(1 + λᵢ/σ²), the bits the i-th principal direction carries through the Gaussian channel.
  4. Sum to the ceiling I_G(σ) = Σᵢ tᵢ and the effective rank d_eff = #{ i : λᵢ ≥ σ² }; report the accessible cap min{ H(e₀), I_G }, since at low noise the discrete entropy H(e₀) binds.

The output is in bits. Beside it the report renders the per-secret readout for the surface (token-F1 on text recovery), per src/talens/report.py. The estimator is src/talens/measures/spectral_channel_mi.py; the geometry-only behaviour is checked model-free in tests/test_spectral_channel_mi.py.

§02

Method

what it estimates · which surface · what it bounds

Kind. A mutual-information ceiling (converse), not an estimate of realized leakage. I_G is the capacity of the additive white Gaussian channel matched to the released embedding, so by the data-processing inequality it upper-bounds I(X; Y) for the true secret X.

Surface. The pooled sentence embedding under a Gaussian release mechanism is the matched surface (Vec2Text, Stained Glass). It also applies to a single released residual vector (Rep2Text), where it is provably non-binding (see Rationale). It is defined per release noise σ, so it has a reading at each sweep point.

What it bounds. Through Fano, the uniform-prior recovery satisfies P(x̂ = X) ≤ (min{H(e₀), I_G} + 1) / H(X) for every attack; the per-mode profile tᵢ and the effective rank d_eff localize the leakage onto the top eigendirections (a water-filling decomposition). The full converse, sufficiency identity, and localization bound are stated and verified on the Vec2Text page.

§03

Rationale

why attack-independent · where it tracks · where it is vacuous

Attack-independent. The probe reads only Σ and σ. It never runs an inverter and never observes the adversary's output, so its correlation with recovery is not circular: it could be computed with the attack code deleted.

Where it tracks recovery. Across the Gaussian-DP budget sweep on pooled GTR embeddings, the ceiling rank-predicts Vec2Text recovery at Spearman ρ = +1.00 on token-F1 (+0.71 on the harder exact-match readout), matching the learned variational bound at roughly 28 times lower cost. Under split-inference perturbation it tracks the best inverter at ρ = +0.958 over the full sweep. See the Vec2Text and split-inference pages.

Where it is provably non-binding (capacity slack). On a single released residual vector (Rep2Text), the ceiling is enormous (about 2856 bits, constant across passage length) while recovery rises with length: ρ ≈ 0.18 over length. The single vector is not a binding capacity bottleneck, so the ceiling has large slack and does not order recovery. This is a true measurement-loop outcome, not a refutation; it is the capacity-slack arm of the probe-failure dichotomy.

Where it is shape-blind. Under the Stained Glass mechanism at a fixed bit budget, the scalar capacity is constant (826.8 bits) while recovery varies twelvefold across noise shapes: ρ(I_G, token-F1) = +0.48 across shapes. The matched predictor there is read-subspace distortion, not the scalar budget. The scalar ceiling is correct within a shape but blind across shapes (see Stained Glass).
